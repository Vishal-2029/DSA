# DSA Learning Plan (90 Days)

This repository is a **90-day DSA learning roadmap** designed to help you build strong fundamentals and solve more problems consistently.

## Purpose
- Learn Data Structures & Algorithms step-by-step.
- Practice **2 DSA questions every day**.
- Track progress day-wise so you always know what to study next.

> Repo goal: **learn DSA**, not just memorize questions.

---

## How to Use This Repo
1. **Follow the day-wise topics below.**
2. For each day:
   - Study the topic(s) mentioned.
   - Solve **2 problems for that day**.
   - Review solutions and note patterns (time complexity, edge cases, recurrence, etc.).
3. Keep a short daily log (optional): what you learned + where you struggled.

## Suggested Workflow for Daily Practice
- **Understand** the topic (concept + common patterns).
- **Solve 2 problems** aligned with the topic.
- **Validate** with constraints:
  - time complexity target (e.g., O(n log n), O(n), etc.)
  - space complexity target
- **Revise**: if you redo a day next week, aim to solve faster with fewer mistakes.
